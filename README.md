## About the Program

In this program, I have used a stack to reverse a given string.
The idea is to push all characters of the string into the stack and then pop them to get the reversed string.

## Concepts Used

* Stack using array
* Push and Pop operations
* Basic string handling

## How it works

1. The user enters a string.
2. Each character is stored in the stack.
3. Characters are removed from the stack one by one.
4. The reversed string is printed.

 Example

Input: Hello
Output: olleH
